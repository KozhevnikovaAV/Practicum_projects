# АНАЛИЗ КНИЖНОЙ БАЗЫ ДАННЫХ ДЛЯ СОЗДАНИЯ ПРИЛОЖЕНИЕ (SQL)   

## ЗАДАЧА:    
Проанализировать базу данных и ответить на вопросы:   
- сколько книг вышло после 1 января 2000 года?  
- сколько обзоров и какова средняя оценкв для каждой книги?   
- какое издательство, выпустило наибольшее число книг (толще 50 страниц)?   
- у какого автора самая высокая средняя оценка книг?   
- каково среднее количество обзоров от пользователей, которые поставили больше 48 оценок?

## ДАННЫЕ:   
* таблица, содержащая данные о книгах (books):   
— идентификатор книги;   
— идентификатор автора;   
— название книги;   
— количество страниц;   
— дата публикации книги;   
— идентификатор издателя.   

* таблица, содержащая данные об авторах (authors):    
— идентификатор автора;   
— имя автора.   

* таблица, содержащая данные об издательствах (publishers):     
— идентификатор издательства;   
— название издательства.   

* таблица, содержащая данные о пользовательских оценках книг (ratings):   
— идентификатор оценки;  
— идентификатор книги;   
— имя пользователя, оставившего оценку;   
— оценка книги.   

* таблица, содержащая данные о пользовательских обзорах (reviews):     
— идентификатор обзора;   
— идентификатор книги;   
— имя автора обзора;   
— текст обзора.
