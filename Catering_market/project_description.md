# ИССЛЕДОВАНИЕ РЫНКА ЗАВЕДЕНИЙ ОБЩЕСТВЕННОГО ПИТАНИЯ Г. МОСКВЫ   

## ЗАДАЧА:   
Проанализировать ситуацию на рынке общественного питания в г. Москве, на основе открытых источников, выделить особенности и презентовать полученные результаты инвесторам из фонда "Shut Up and Take My Money"   

## ДАННЫЕ:   
Доступен файл (moscow_places.csv)  на основе данных сервисов Яндекс Карты и Яндекс Бизнес на лето 2022 года:   
— название заведения;   
— адрес заведения;   
— категория заведения (например «кафе», «пиццерия» или «кофейня»);   
— информация о днях и часах работы;   
— широта географической точки, в которой находится заведение;   
— долгота географической точки, в которой находится заведение;   
— рейтинг заведения по оценкам пользователей в Яндекс Картах (высшая оценка — 5.0);   
— категория цен в заведении (например «средние», «ниже среднего», «выше среднего» и так далее);   
— строка, которая хранит среднюю стоимость заказа в виде диапазона (например: «Средний счёт: 1000–1500 ₽»; «Цена чашки капучино: 130–220 ₽»; «Цена бокала пива: 400–600 ₽» и так далее);   
— число с оценкой среднего чека, которое указано только для значений из столбца о средней стоимости заказа в виде диапазона;   
— число с оценкой одной чашки капучино, которое указано только для значений из о средней стоимости заказа в виде диапазона;   
— число, выраженное 0 или 1, которое показывает, является ли заведение сетевым (0 - несетевое, 1 - сетевое);     
— административный район, в котором находится заведение (например Центральный административный округ);   
— количество посадочных мест.   

## ИСПОЛЬЗУЕМЫЕ БИБЛИОТЕКИ:   
pandas, numpy, matplotlib, seaborn, json, plotly.express
