# МУЗЫКА БОЛЬШИХ ГОРОДОВ (вводный проект)   

## ЗАДАЧА:    
Сравнить музыкальные предпочтения пользователей сервиса "Янедекс.Музыка" из г. Москвы и г. Санкт-Петербурга.    
Сравнить прослушивания пользователей городов по дням недели: понедельникЮ среда и пятница. Также провести сравнение среди жанров музыки, которые пользователи из городов предпочитают по утрам в понедельник и по вечерам в пятницу.   

## ОПИСАНИЕ:   
Файл с данными (yandex_music_project.csv) содержит следующую информацию:   
— идентификатор пользователя;   
— название трека;   
— имя исполнителя;   
— название жанра;   
— город пользователя;   
— время начала прослушивания;   
— день недели.
