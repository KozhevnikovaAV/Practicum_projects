# ПРОГНОЗИРОВАНИЕ ПРОДАЖ ДЛЯ ИНТЕРНЕТ-МАГАЗИНА "Стримчик" НА 2017 ГОД   

## ЗАДАЧА:   
Определить закономерность успешности игры, сделать ставку на потенциально популярный продукт, составить портреыт пользователей для регионов, проверить две гипотезы (1. Средние пользовательские рейтинги платформ Xbox One и PC одинаковые; 2. Средние пользовательские рейтинги жанров Action и Sports разные) и спланировать рекламные кампании.

## ДАННЫЕ:   
Файл содержит информацию об играх до 2016 года (games.csv):   
— название игры;   
— платформа;   
— год выпуска;   
— жанр игры;   
— продажи в Северной Америке (миллионы проданных копий);    
— продажи в Европе (миллионы проданных копий);    
— продажи в Японии (миллионы проданных копий);    
— продажи в других странах (миллионы проданных копий);    
— оценка критиков (максимум 100);    
— оценка пользователей (максимум 10);   
— рейтинг от организации ESRB (англ. Entertainment Software Rating Board).   

## ИСПОЛЬЗУЕМЫЕ БИБЛИОТЕКИ:   
pandas, matplotlib.pyplot, numpy, seaborn, scipy
